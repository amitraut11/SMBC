# Login Card Button

The `LoginCardButton` component is a customizable button designed for login forms. It can be styled with different colors and labels.

## Usage

You can use the `LoginCardButton` like this:

```tsx
<LoginCardButton 
  backgroundColor="#28a745" 
  textColor="#fff" 
  label="Go" 
/>
