# Footer

The `Footer` component displays contact information and navigation links, with customizable background and text colors.

## Usage

You can use the `Footer` like this:

```tsx
<Footer 
  email="info@xyzbank.com" 
  year={2024} 
  backgroundColor="#3b82f6" 
  textColor="white" 
/>
