# Login Card Input

The `LoginCardInput` component is a customizable input field designed for login forms. You can adjust its placeholder, background color, border color, and text color.

## Usage

You can use the `LoginCardInput` like this:

```tsx
<LoginCardInput 
  placeholder="Enter username" 
  backgroundColor="#fff" 
  borderColor="#ccc" 
  textColor="#000" 
/>
