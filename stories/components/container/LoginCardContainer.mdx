# Login Card Container

The `LoginCardContainer` component is designed to wrap login-related components, providing a customizable background color, width, and padding.

## Usage

You can use the `LoginCardContainer` like this:

```tsx
<LoginCardContainer 
  backgroundColor="#333" 
  width="300px" 
  padding="20px"
>
  {/* Your login form components will go here */}
</LoginCardContainer>
